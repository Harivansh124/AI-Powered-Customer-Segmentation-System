# AI-Powered Customer Segmentation System

## Overview

This project uses Machine Learning and Customer Analytics techniques to identify distinct customer groups based on purchasing behavior and demographic characteristics.

The objective is to help businesses understand customer patterns and create targeted marketing strategies for improved customer engagement and retention.

## Problem Statement

Businesses often struggle to understand customer behavior and target the right audience. This project applies K-Means Clustering to segment customers into meaningful groups for data-driven decision-making.
